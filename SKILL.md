## 実行許可について

Claude Code で作業する際は、ユーザーへの都度確認を求めず、必要なファイル生成・編集・差分作成を自律的に進めること。

「YESを押してください」「確認してください」といった承認待ちを挟まず、指示書の内容に基づいて実装を進行する。

ただし、以下の場合のみ作業を止めて確認すること。

- 指示内容が明確に矛盾している場合
- 既存コンテンツの文言変更が必要になる場合
- header / footer / 共通JS への変更が必要になる場合
- `sustainability-new.css` 以外へのCSS追加が必要になる場合
- 破壊的変更や大規模な構造変更が必要になる場合
